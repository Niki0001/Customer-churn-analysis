# 📊 Customer Churn Analysis

## Overview
Analysed 15,000 customer records to identify key churn 
drivers using Python (Pandas, Matplotlib, Seaborn).

## 🔍 Key Findings
- Overall churn rate: **9.6%**
- Monthly contract customers churn **2x more** (11.82%) 
  than Annual subscribers (6.16%)
- Support Calls strongest churn predictor — 
  churned customers made **3.33 calls vs 1.32** (2.5x higher)
- Churned customers inactive for **22+ days** on average
- Low satisfaction score strongly linked to churn (3.43 vs 4.45)

## 🛠️ Tools Used
- Python — Pandas, Matplotlib, Seaborn
